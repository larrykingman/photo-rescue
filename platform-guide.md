## Photoshop 使用指南

### 核心功能

#### 1. 生成式填充（Generative Fill）

**使用场景**：
- 清除路人/杂物
- 扩展背景
- 添加元素

**操作步骤**：
```
1. 选择需要处理的区域（套索工具/矩形选框）
2. 右键 → 生成式填充
3. 输入提示词（支持中英文）
4. 点击生成，选择最佳结果
```

**提示词技巧**：
```
✅ 好的提示词：
"Remove all strangers, reconstruct clean wall with natural texture"
"删除所有路人，恢复干净的墙面，纹理自然"

❌ 差的提示词：
"Make it look better"（太模糊）
"好看一点"（不具体）
```

**参数建议**：
- 选区不要太紧，留出边缘过渡空间
- 提示词简洁明了，50字以内
- 多次生成，选择最佳结果

---

#### 2. 神经滤镜（Neural Filters）

**使用场景**：
- 智能肖像（表情调整）
- 皮肤平滑
- 超级缩放

**操作路径**：
```
滤镜 → Neural Filters
```

**智能肖像功能**：
- 调整表情（快乐、惊讶、愤怒）
- 调整头部方向
- 调整光照方向
- 调整年龄

---

#### 3. 调整图层

**常用调整**：
```
图像 → 调整：
- 曝光度（调整整体亮度）
- 阴影/高光（恢复细节）
- 曲线（调整对比度）
- 色相/饱和度（调整颜色）
```

**非破坏性编辑**：
- 使用调整图层而非直接调整
- 可随时修改参数
- 可调整图层不透明度

---

#### 4. 修复画笔工具

**使用场景**：
- 去除小瑕疵
- 修复皮肤问题
- 平滑过渡

**工具选择**：
```
修复画笔（J）: 适合小面积修复
仿制图章（S）: 适合大面积复制
内容感知移动: 适合移动物体
```

---

### 快捷键速查

```
Ctrl+J: 复制图层
Ctrl+Z: 撤销
[: 减小画笔大小
]: 增大画笔大小
空格: 临时抓手工具
Ctrl+T: 自由变换
Ctrl+M: 曲线
Ctrl+L: 色阶
```

## Midjourney 使用指南

### 图生图（Image-to-Image）

**基本语法**：
```
/imagine prompt: [图片URL] [提示词] --参数
```

---

### 核心参数

#### 1. 图片权重 (--iw)

**作用**：控制与原图的相似度

**取值范围**：0-3（默认1）

**建议值**：
```
严格保持主体：--iw 2.5-3.0
适度改变风格：--iw 1.5-2.0
创意重构：--iw 0.5-1.0
```

**示例**：
```
/imagine prompt: 
https://example.com/photo.jpg
portrait, professional lighting, clean background
--iw 2.5 --style raw
```

---

#### 2. 负面提示词 (--no)

**作用**：排除不需要的元素

**示例**：
```
--no strangers, crowd, watermark, text
--no 路人, 水印, 文字
```

---

#### 3. 风格参数

```
--style raw     # 真实摄影风格
--style scenic  # 风景风格
--v 6.0        # 使用V6模型（推荐）
```

---

#### 4. 纵横比 (--ar)

```
--ar 1:1   # 正方形
--ar 4:5   # 人像常用
--ar 16:9  # 风景/视频
--ar 9:16  # 竖屏
```

---

### 提示词结构

**推荐结构**：
```
[主体描述] + [修复内容] + [风格要求] + [参数]

示例：
/imagine prompt:
https://example.com/photo.jpg
Woman with natural smile, clean background, professional lighting
--iw 2.5 --no strangers --ar 4:5 --style raw
```

---

### 局部重绘（Vary Region）

**使用场景**：只修改画面的一部分

**操作步骤**：
```
1. 生成图片后，点击 "Vary (Region)"
2. 使用矩形或套索工具选择区域
3. 在提示词中描述修改内容
4. 点击生成
```

**示例**：
```
选择背景区域
提示词："Clean blue sky, no clouds"
→ 只改变天空，人物不变
```

---

### 多次迭代策略

**提高成功率的技巧**：
```
1. 先生成基础版本
2. 使用 V1/V2/V3/V4 选择最佳变体
3. 使用 Vary (Subtle) 微调
4. 使用 Vary (Strong) 大幅调整
5. 多次尝试，选择最佳结果
```

## 即梦AI/豆包 使用指南

### 核心功能

#### 1. 图生图

**操作步骤**：
```
1. 上传原图
2. 选择"图生图"模式
3. 输入提示词
4. 设置参考图强度
5. 生成并选择最佳
```

---

#### 2. 提示词格式

**推荐格式**：
```
【主体保留】[详细描述人物特征]
【修复内容】[具体修复描述]
【禁止变异】[列出禁止改变的内容]
参考图强度：0.85-0.9
```

**示例**：
```
【主体保留】
年轻女性，长发，白色上衣，牛仔裤
站立姿势，面朝镜头，自然微笑

【修复内容】
提亮面部光影，去除背景路人
增加正面柔光，优化肤色

【禁止变异】
禁止改变面部特征、身材、服装
禁止改变姿势和位置

参考图强度：0.88
```

---

#### 3. 参考图强度

**取值范围**：0-1

**建议值**：
```
0.9-0.95: 极高一致性，微调为主
0.85-0.9: 高一致性，适度修改（推荐）
0.7-0.85: 中等一致性，较大修改
```

**调整原则**：
- 主体需要保持一致：强度 ≥ 0.85
- 适度修复：强度 0.85-0.9
- 创意改变：强度 0.7-0.85

---

#### 4. 局部重绘

**使用场景**：只修改特定区域

**操作步骤**：
```
1. 上传图片
2. 使用画笔涂抹需要修改的区域
3. 输入修改提示词
4. 生成并选择
```

**示例**：
```
涂抹背景区域
提示词："干净的蓝天白云"
→ 只改变背景
```

---

### 中文提示词技巧

**优势**：即梦AI/豆包对中文提示词理解较好

**建议**：
```
✅ 使用具体的中文描述：
"将上衣改为深蓝色，去除褶皱"
"提亮面部，添加正面柔光"

✅ 使用结构化格式：
【主体保留】...
【修复内容】...
【禁止变异】...

❌ 避免过于口语化：
"好看一点"
"美颜一下"
```

## NanoBanana 使用指南

### 权重标注法

**基本语法**：
```
(关键词:权重值)
```

**权重值范围**：
```
0.5: 降低权重
1.0: 默认权重
1.5: 提高权重
2.0: 最高权重（推荐用于保持主体）
```

---

### 核心参数

#### 1. 强度 (--strength)

**作用**：控制变化程度

**取值范围**：0-1

**建议值**：
```
0.2-0.3: 极小变化，高度保持原图
0.4-0.5: 适度变化
0.6-0.8: 较大变化
```

**示例**：
```
--strength 0.3  # 修复废片推荐值
```

---

#### 2. 质量 (--quality)

```
--quality 1: 快速生成
--quality 2: 标准质量（推荐）
--quality 3: 高质量
```

---

### 提示词结构

**推荐结构**：
```
(保持主体不变:2.0),
(主体描述:1.5),
(修复内容:1.3),
--quality 2 --strength 0.3
```

**示例**：
```
(Keep original face unchanged:2.0),
(Keep pose unchanged:2.0),
(Young woman with natural smile:1.5),
(Clean background:1.3),
(Professional lighting:1.2),
--quality 2 --strength 0.3 --no face change, body change
```

---

### 负面提示词

**语法**：
```
--no [不需要的元素]
```

**示例**：
```
--no face change, body change, different clothing, different pose
--no 变脸, 变身材, 变服装, 变姿势
```

## Lightroom 使用指南

### 核心调整

#### 1. 基础光影调整

**常用参数**：
```
曝光（Exposure）: 调整整体亮度
对比度（Contrast）: 调整明暗对比
高光（Highlights）: 降低过曝
阴影（Shadows）: 提亮暗部
白色色阶（Whites）: 调整最亮区域
黑色色阶（Blacks）: 调整最暗区域
```

**逆光修复示例**：
```
曝光: +1.0
阴影: +40
高光: -30
白色色阶: -20
黑色色阶: +10
```

---

#### 2. 局部调整

**使用工具**：
```
调整画笔: 局部调整特定区域
渐变滤镜: 渐进式调整
径向滤镜: 环形调整区域
```

**操作步骤**：
```
1. 选择调整画笔
2. 涂抹需要调整的区域
3. 调整参数（曝光、阴影等）
4. 可以添加多个画笔，分别调整
```

---

#### 3. 颜色调整

**色温/色调**：
```
色温: 调整冷暖（蓝-黄）
色调: 调整色偏（绿-洋红）
```

**HSL调整**：
```
色相（Hue）: 改变颜色
饱和度（Saturation）: 调整颜色鲜艳度
明度（Luminance）: 调整颜色亮度
```

**改衣服颜色示例**：
```
HSL → 目标颜色（如蓝色）
色相: 调整色相
饱和度: 调整鲜艳度
明度: 调整亮度
```

---

### 预设使用

**创建预设**：
```
1. 调整好一张照片
2. 点击"+"创建预设
3. 勾选要保存的调整项
4. 命名并保存
```

**批量应用**：
```
1. 选择多张照片
2. 应用预设
3. 同步设置
```

---

### 快捷键

```
V: 转换为黑白
\: 显示/隐藏调整前后对比
Y: 并排对比
Ctrl+Shift+R: 重置为默认
```

## 平台选择建议

### 根据问题类型选择

| 问题类型 | 推荐平台 | 原因 |
|---------|---------|------|
| 清除路人 | Photoshop, Midjourney | 生成式填充效果好 |
| 修复光影 | Lightroom, 即梦AI | 专业光影调整 |
| 调整姿势 | Photoshop, Midjourney | 生成式调整精准 |
| 改衣服 | Photoshop, Midjourney | 生成式修改效果好 |
| 构图调整 | Lightroom, Photoshop | 裁剪旋转方便 |
| 添加人物 | Photoshop, Midjourney | 生成式添加自然 |
| 去模糊 | Topaz Sharpen AI, Photoshop | 专业锐化工具 |

---

### 根据技能水平选择

**新手推荐**：
```
即梦AI/豆包: 操作简单，中文友好
醒图/美图秀秀: 手机App，易上手
```

**进阶推荐**：
```
Midjourney: 效果好，学习曲线中等
Lightroom: 专业光影调整
```

**专业推荐**：
```
Photoshop: 精确控制，专业级效果
组合使用: PS + MJ + LR
```

---

### 根据时间预算选择

**快速修复（5分钟）**：
```
即梦AI: 一键生成
手机App: 快速调整
```

**标准修复（15分钟）**：
```
Midjourney: 生成+选择
Lightroom: 光影调整
```

**精细修复（30分钟+）**：
```
Photoshop: 多步骤修复
多平台组合: 发挥各平台优势
```

---

### 多平台组合策略

**推荐组合**：

#### 组合1：光影 + 清理
```
1. Lightroom: 光影基础调整
   ↓
2. Photoshop: 清除路人，修复细节
   ↓
3. Lightroom: 最终色调优化
```

#### 组合2：创意修复
```
1. Midjourney: 创意重构
   ↓
2. Photoshop: 精细调整
   ↓
3. Lightroom: 最终输出
```

#### 组合3：快速批量
```
1. 即梦AI: 批量光影调整
   ↓
2. Photoshop: 重点照片精细修复
```

---

**选择合适的平台，是高效拯救废片的关键。根据具体问题、技能水平和时间预算，灵活选择最适合的工具。**
