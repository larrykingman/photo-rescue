## 标准修复流程

### 总体流程图

```
照片输入
    ↓
【阶段1】诊断问题（2分钟）
    ├─ 识别问题类型
    ├─ 评估严重程度
    └─ 确定修复优先级
    ↓
【阶段2】制定策略（1分钟）
    ├─ 选择修复工具
    ├─ 确定修复顺序
    └─ 准备提示词
    ↓
【阶段3】执行修复（5-15分钟）
    ├─ 按优先级逐项修复
    ├─ 保存中间版本
    └─ 验证修复效果
    ↓
【阶段4】质量检查（2分钟）
    ├─ 检查主体一致性
    ├─ 检查光影自然度
    └─ 检查细节完整度
    ↓
【阶段5】输出交付
    ├─ 提供修复后照片
    ├─ 提供修复说明
    └─ 提供预防建议
```

### 时间分配建议

| 阶段 | 时间占比 | 建议 |
|------|----------|------|
| 诊断问题 | 10% | 不要跳过，准确诊断是基础 |
| 制定策略 | 5% | 快速决策，选择最佳方案 |
| 执行修复 | 75% | 主要工作时间，分步进行 |
| 质量检查 | 10% | 必须进行，避免过度修复 |

## 优先级排序原则

### 注意力权重机制

```
修复优先级 = 问题影响范围 × 严重程度
```

### 五层修复顺序（按影响力排序）

#### 第1层：光影修复（最高优先级）⭐⭐⭐

**原因**：光影决定照片是否可用

**何时执行**：
- 逆光导致面部死黑
- 过曝/过暗导致细节丢失
- 阴阳脸影响观感

**操作要点**：
- 先做全局曝光调整
- 再做局部光影优化
- 最后添加修饰光效

#### 第2层：清除干扰 ⭐⭐⭐

**原因**：路人、杂物严重影响画面

**何时执行**：
- 有明显路人/杂物
- 背景杂乱干扰主体
- 画面有不需要的元素

**操作要点**：
- 先清除大型干扰物
- 再处理细节瑕疵
- 保持背景自然

#### 第3层：修复主体 ⭐⭐

**原因**：主体是照片核心

**何时执行**：
- 表情/姿势有问题
- 衣服不满意
- 人物特征需要调整

**操作要点**：
- 先修表情，再调姿势
- 最后调整服装细节
- 保持身份一致性

#### 第4层：构图调整 ⭐

**原因**：构图影响整体美感

**何时执行**：
- 照片歪斜
- 主体位置不佳
- 比例不理想

**操作要点**：
- 先旋转校正
- 再裁剪构图
- 考虑输出用途

#### 第5层：细节优化 ⭐

**原因**：锦上添花

**何时执行**：
- 轻度模糊需要锐化
- 颜色需要微调
- 细节需要增强

**操作要点**：
- 适度锐化，避免过度
- 保持自然感
- 不破坏前期修复

## 分步修复指南

### 问题组合修复策略

#### 场景1：路人 + 逆光

**推荐顺序**：
```
1. 修复逆光（先让照片可用）
   ↓
2. 清除路人（恢复干净背景）
   ↓
3. 质量检查
```

**原因**：光影问题影响整体，必须先解决

---

#### 场景2：姿势丑 + 衣服褶皱 + 逆光

**推荐顺序**：
```
1. 修复逆光（优先级最高）
   ↓
2. 调整姿势（主体修复）
   ↓
3. 去除衣服褶皱（细节优化）
   ↓
4. 整体质检
```

---

#### 场景3：模糊 + 歪斜 + 路人

**推荐顺序**：
```
1. 清除路人（影响构图判断）
   ↓
2. 旋转校正（调整构图）
   ↓
3. 裁剪优化
   ↓
4. 锐化去模糊（最后执行）
```

**原因**：锐化应在构图确定后进行，避免重复处理

---

#### 场景4：少人 + 光影 + 表情

**推荐顺序**：
```
1. 修复光影（基础优先）
   ↓
2. 添加缺失人物（重建场景）
   ↓
3. 修复表情（微调细节）
   ↓
4. 整体协调性检查
```

---

### 单问题深度修复

#### 光影修复详细步骤

```
步骤1：分析光影
├─ 识别光源方向
├─ 评估曝光情况
└─ 标记问题区域

步骤2：全局调整
├─ 调整曝光（整体亮度）
├─ 调整对比度
└─ 调整高光/阴影

步骤3：局部优化
├─ 提亮面部阴影
├─ 降低过曝高光
└─ 平衡阴阳脸

步骤4：质感增强
├─ 添加眼神光
├─ 增强皮肤质感
└─ 优化轮廓光

步骤5：效果验证
├─ 检查是否自然
├─ 检查细节保留
└─ 检查氛围一致
```

#### 姿势修复详细步骤

```
步骤1：识别问题姿势
├─ 标记僵硬部位
├─ 识别不协调姿态
└─ 确定调整方向

步骤2：制定调整方案
├─ 脊柱矫正
├─ 手臂位置调整
├─ 重心调整
└─ 表情优化

步骤3：生成提示词
├─ 明确调整内容
├─ 强调保留约束
└─ 添加负面提示

步骤4：执行修复
├─ 局部调整优先
├─ 逐步微调
└─ 保存中间版本

步骤5：效果验证
├─ 检查姿势自然度
├─ 检查身份一致性
└─ 检查光影匹配
```

## 平台操作指南

### Photoshop 操作流程

#### 准备工作
1. 打开照片
2. 复制背景图层（Ctrl+J）
3. 保存为PSD文件

#### 修复流程

**步骤1：光影调整**
```
图像 → 调整 → 曝光度
图像 → 调整 → 阴影/高光
使用曲线调整对比度
```

**步骤2：清除干扰**
```
选择 → 主体（自动选择）
反选（Ctrl+Shift+I）
编辑 → 生成式填充
输入提示词：删除所有路人，恢复干净背景
```

**步骤3：修复主体**
```
选择需要调整的部位
编辑 → 生成式填充
输入具体提示词
```

**步骤4：细节优化**
```
滤镜 → 锐化 → 智能锐化
滤镜 → 杂色 → 减少杂色
```

**步骤5：质量检查**
```
放大检查细节
对比原图检查一致性
调整图层不透明度微调效果
```

### Midjourney 操作流程

#### 图生图修复

**步骤1：上传原图**
```
/discord
上传照片到Midjourney
复制图片URL
```

**步骤2：生成修复提示词**
```
/imagine prompt:
[原图URL]
[主体描述], [修复内容描述]
--iw 2.0 --style raw --ar [比例]
--no [不需要的元素]
```

**步骤3：调整参数**
```
--iw 2.0-3.0  # 确保主体一致
--style raw   # 真实风格
--no ...      # 负面提示词
```

**步骤4：选择最佳结果**
```
U1/U2/U3/U4  # 放大选择
V1/V2/V3/V4  # 变体调整
```

---

### 即梦AI/豆包 操作流程

#### 图生图修复

**步骤1：上传原图**
- 选择"图生图"模式
- 上传照片

**步骤2：输入提示词**
```
【主体保留】严格保持人物外貌、姿势、服装不变
【修复内容】[具体修复描述]
【禁止变异】[列出禁止改变的内容]
参考图强度：0.85-0.9
```

**步骤3：调整参数**
- 参考图强度：0.85-0.9（确保一致性）
- 生成多个版本，选择最佳

**步骤4：检查结果**
- 对比原图检查主体一致性
- 检查修复效果
- 必要时重新生成

### Lightroom 操作流程

#### 光影调整为主

**步骤1：基础调整**
```
曝光：+/- [档数]
对比度：+/- [数值]
高光：- [数值]
阴影：+ [数值]
白色色阶：调整
黑色色阶：调整
```

**步骤2：局部调整**
```
使用调整画笔
提亮面部阴影
降低高光
```

**步骤3：颜色调整**
```
色温：调整冷暖
色调：调整色偏
饱和度：适度调整
```

**步骤4：导出**
```
文件 → 导出
选择格式和质量
```

## 常见问题处理

### 问题1：修复后主体变形

**原因**：提示词不够明确，或参考图强度不够

**解决方案**：
```
1. 提高参考图强度（0.9以上）
2. 在提示词中添加：
   "Keep original face unchanged"
   "Maintain original body shape"
   "NO face change, NO body change"
3. 使用负面提示词：
   "--no face change, body change"
```

---

### 问题2：背景修复不自然

**原因**：背景复杂，AI填充不理想

**解决方案**：
```
1. 先简化提示词：
   "Clean background, simple texture"
2. 多次生成，选择最佳
3. 局部修复：
   - 分区域清除路人
   - 分步骤修复背景
4. 手动调整：
   - 使用Photoshop修复画笔
   - 手动融合边缘
```

---

### 问题3：光影修复后不自然

**原因**：过度调整，失去真实感

**解决方案**：
```
1. 降低调整强度
2. 分步调整：
   - 先调整体曝光
   - 再调局部光影
3. 保持适度对比度
4. 参考自然光影参考图
5. 最后添加质感细节
```

---

### 问题4：修复后失真严重

**原因**：一次性修复过多问题

**解决方案**：
```
1. 采用分步修复：
   - 每次只修一个问题
   - 保存中间版本
2. 降低AI强度：
   - Midjourney: --iw 提高到3.0
   - 即梦AI: 参考图强度 0.9
3. 手动微调：
   - 使用Photoshop调整图层
   - 调整不透明度
```

---

### 问题5：添加的人物不协调

**原因**：光影、风格不匹配

**解决方案**：
```
1. 详细描述人物特征：
   - 性别、年龄、身高
   - 发型、服装
   - 姿势、表情
2. 强调光影匹配：
   "Match existing lighting direction"
   "Same color temperature"
   "Consistent shadow softness"
3. 多次生成，选择最佳
4. 后期微调：
   - 调整色温一致
   - 调整光影方向
```

## 质量检查清单

### 修复完成后的检查项

```
✅ 主体一致性检查
  [ ] 面部特征是否一致？
  [ ] 身材比例是否自然？
  [ ] 服装是否保持？
  [ ] 姿势是否自然？

✅ 光影自然度检查
  [ ] 光影方向是否统一？
  [ ] 色温是否自然？
  [ ] 阴影是否合理？
  [ ] 高光是否过度？

✅ 细节完整度检查
  [ ] 边缘是否自然？
  [ ] 纹理是否保留？
  [ ] 是否有瑕疵？
  [ ] 是否有AI痕迹？

✅ 整体协调性检查
  [ ] 主体与背景是否协调？
  [ ] 色彩是否和谐？
  [ ] 构图是否平衡？
  [ ] 整体是否自然？
```

## 效率提升技巧

### 批量修复

**适用场景**：多张照片有相同问题

**操作方法**：
1. Photoshop：使用动作录制，批量执行
2. Lightroom：预设同步调整
3. AI工具：一次上传多张，批量生成

### 模板化修复

**创建提示词模板**：
```
# 清除路人模板
Remove all strangers from background. 
Use generative fill to reconstruct [background type].
Keep main subject unchanged.

# 逆光修复模板
Brighten face by [X] stops. 
Add soft fill light from front.
Reduce backlight harshness.
Increase shadow detail by [X]%.
```

### 快捷键使用

**Photoshop常用快捷键**：
```
Ctrl+J: 复制图层
Ctrl+Z: 撤销
Ctrl+Shift+Z: 重做
[: 减小画笔
]: 增大画笔
Space: 临时抓手工具
```
